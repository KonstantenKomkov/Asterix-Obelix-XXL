# Runtime asset pipeline v1

The Gaul stage-one runtime package is built from an installed XXL1 data set with one command:

```sh
scripts/build_slice_assets.sh /path/to/game /path/to/gaul-stage-1.astpak
```

The command extracts a temporary, portable importer proof and packages it as the deterministic ASTPAK 1.0 format. Existing output is never overwritten.

Resources include canonical JSON meshes, animations and skins, WAV audio, a scene manifest, and textures encoded as `ASTMTEX` version 1. Texture payloads use tightly packed `rgba8Unorm` pixels and contain every mip level down to 1×1. Mips are generated by a deterministic 2×2 box filter; odd edges average only available pixels.

`ASTMTEX` starts with the eight bytes `ASTMTEX\n`, followed by little-endian `version`, `pixelFormat` (`1` = RGBA8Unorm), `mipCount`, and `dataOffset` uint32 values. Each mip table entry stores `width`, `height`, relative `offset`, and `length` as uint32 values.
