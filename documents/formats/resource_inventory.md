# Инвентаризация оригинальных файлов

Инвентаризатор читает локальную установленную копию игры напрямую. Запуск игры, видео и захват экрана для этого не требуются.

```sh
make inventory GAME_DIR="/path/to/AsterixXXL" OUTPUT="$HOME/asterix-reference/metadata/inventory.json"
```

Если `OUTPUT` не задан, JSON выводится в stdout. Рекомендуется хранить полный манифест вне репозитория: он содержит имена, размеры и хеши файлов конкретной копии игры.

## Схема v1

Корень содержит `schemaVersion`, `fileCount`, `totalSize` и отсортированный массив `files`. Для каждого файла записываются:

- `path` — относительный путь с `/` в качестве разделителя;
- `size` — размер в байтах;
- `sha256` — SHA-256 полного содержимого;
- `signature` — первые 16 байт в hex (или весь файл, если он короче);
- `extension` — расширение в нижнем регистре;
- `level` — номер для файлов внутри каталога `LVLnnn`.

В манифест намеренно не входят абсолютный путь и время генерации, поэтому повторный запуск на неизменной копии создаёт побайтно идентичный JSON.
