# Visual regression запуска Gaul

**Проверено:** 22 июля 2026.

Regression фиксирует первый стабильный gameplay-кадр после загрузки Gaul Stage
1: content area окна 1280×720 points, отсутствие ввода, камера уже сопровождает
стартовый spawn, Астерикс находится в idle. HUD оставлен в кадре и дополнительно
подтверждает idle, здоровье, checkpoint и FOV. Эталон и проверяемый
PNG хранятся локально вне Git рядом с производным ASTPAK.

## Захват

1. Установить актуальный пакет через `scripts/install_slice_assets.sh`.
2. Запустить debug-приложение, выбрать New Game и дождаться 60 отрисованных
   кадров без ввода.
3. Захватить только Metal viewport в PNG без масштабирования, color-profile
   conversion. Размер обязан быть 1280×720 физических пикселей либо
   2560×1440 на Retina; reference и actual должны совпадать по размеру.
4. Первый принятый кадр сохранить как
   `~/asterix-reference/gaul-launch/reference.png`; повторный запуск — как
   `actual.png`.

Проверка выполняется одной командой:

```sh
make visual-regression \
  REFERENCE="$HOME/asterix-reference/gaul-launch/reference.png" \
  ACTUAL="$HOME/asterix-reference/gaul-launch/actual.png"
```

Comparator декодирует PNG без ресайза и принимает одинаковые 16:9 кадры не
меньше 1280×720. Он отклоняет пустой/однотонный render,
среднюю абсолютную RGB-ошибку выше 3,5% либо более 12% пикселей, у которых хотя
бы один канал изменился сильнее 24/255. Для центральной области персонажа
отдельный порог изменённых пикселей — 2,5%. Поэтому крупные изменения силуэта
персонажа (включая T-pose), стартовой проекции/spawn и потерянные texture
bindings не проходят, а небольшие различия Metal rasterization допускаются.

## Локальная приёмка

Эталон и второй кадр сняты с одного и того же локального Gaul ASTPAK после
холодного запуска. Автоматическая сверка прошла; кадры и пакет не добавлены в
Git согласно политике распространения ресурсов. Синтетические unit-тесты
закрепляют успешное совпадение, отказ на крупном изменении и защиту от ложного
эталона с пустым render.
