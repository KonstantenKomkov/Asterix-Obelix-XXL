# Interactive world runtime

`interactive::Runtime` хранит authoritative fixed-tick состояние интерактивов
вертикального среза: volume triggers, рычагов, разрушаемых объектов, связанных с
ними наград и checkpoint. Все сущности имеют устойчивые ненулевые ID, а изменения
публикуются типизированными событиями.

Триггер срабатывает на входе в AABB и поддерживает one-shot/repeatable режим.
Рычаг активируется фронтом action `interact` в заданном радиусе. Для клавиатуры
по умолчанию используется `E`, для gamepad — B. Разрушаемый объект получает урон
из общей combat hitbox-системы; при нулевом здоровье он открывает связанную
награду, которая автоматически подбирается игроком.

Активация checkpoint сохраняет копию mutable world state после всех изменений
этого simulation tick. При падении capsule runtime возвращает игрока к checkpoint
и одновременно откатывает интерактивы, награды, противника и combat state. После
смерти тот же rollback и восстановление полного здоровья запускаются action
`interact`. Поэтому собранные после checkpoint награды не дублируются, а
разрушенные после него объекты и активированные рычаги корректно возвращаются.

Metal integration пока создаёт минимальный синтетический набор рядом со spawn:
trigger, lever, destructible, reward и checkpoint. Их state/counters входят в
диагностический snapshot и HUD. Оригинальные модели, скрипты или производные
ресурсы для этого не используются; привязка импортированных entity definitions
остаётся отдельным content-проходом.
